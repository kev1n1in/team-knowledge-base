# 團隊知識庫 Team Knowledge Base

這個 repo 是團隊的共享知識庫，用來記錄技術文件、工作流程、最佳實踐，以及任何有助於團隊協作的資訊。

## 這個知識庫是做什麼的

- 統一存放技術文件與教學筆記
- 記錄常見問題與解決方案（FAQ）
- 分享工具使用指南與工作流程
- 讓新成員快速上手團隊慣例

## 安裝方式

### 1. Clone 這個 repo

```bash
git clone https://github.com/kev1n1in/team-knowledge-base.git
cd team-knowledge-base
```

### 2. 開始貢獻

直接編輯或新增 Markdown 文件（`.md`），然後提交：

```bash
git add .
git commit -m "新增：你的文件描述"
git push origin main
```

### 3. 建議的目錄結構

```
team-knowledge-base/
├── README.md          # 本文件
├── guides/            # 工具與流程指南
├── faq/               # 常見問題
└── onboarding/        # 新人上手資料
```

## 貢獻規範

- 文件使用 Markdown 格式
- 檔名使用小寫英文加連字號（e.g., `setup-guide.md`）
- 每次更新請寫清楚 commit message

## 聯絡

有任何問題，請直接在 GitHub Issues 提出。
