Agentic AI

定義：具備「自主性」的 AI 系統。它不再只是等你問、它答，而是會根據目標「自我拆解步驟」。

思維轉變：從「給我一個答案」變成「幫我達成一個目標」。

關鍵能力：推理 (Reasoning) + 規劃 (Planning) + 自我修正 (Self-Correction)。

那為什麼「不要用」？

成本

Token
Agent 每思考一步都要燒錢。複雜任務可能反覆思考 20 次，成本是單次對話的數十倍。
時間
自主思考與工具調用需要時間，Agent 通常比直覺回答慢得多。無法清楚明確表達步驟的需求可能讓 Agent 一直像弱智一樣鬼打牆。
風險
資料不小心外流或原本 vibe 出來的工具不小心一次遺失過去所有儲存的客戶資料，得不償失。

Agents 的武裝實作

Agent
士兵卡
被派出去執行獨立任務的 AI 子代理。可以並行、可以背景執行，不需要你盯著。
Skills
裝備
封裝好的能力模組。Agent 可以呼叫 Skills 完成特定動作，像士兵帶著不同武器。
Claude Rules
戰場守則
CLAUDE.md 裡定義的規則。每次對話自動生效，不需要重複說明，像作戰手冊。
Hooks
觸發卡
事件發生時自動觸發的行為。存檔後自動做某件事，你不需要記得叫它做。
